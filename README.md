# cybersecurity-homelab

## Overview

This repository documents my cybersecurity home lab, where i build and practice real-world skills in digital forensics, Linux administration, networking and incident respone.

The goal of this lab is to create an enterprise-style environment for hands-on learning and project development.

---

## Objective

- Build multiple virtual machines
- Practice digital forensics
- Learn Linux administration
- Analyze network traffic
- Develop Bash and Python scripts
- Simulate incident response scenarios
